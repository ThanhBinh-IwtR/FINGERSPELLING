Our Project

Problem

Currently more than 1.5 billion people (nearly 20% of the global population) live with hearing loss

70 million Deaf people around the world who use sign language to communicate

AI solutions include automated speech recognition (ASR) and machine translation, These technologies are often not accessible to them.

Solve

Fingerspelling uses hand shapes that represent individual letters to convey words

Develop a system that recognizes sign language through fingers.

Strategies

Planing

Specify project goals, audience, and website purpose. Define desired features and functionalities.

Data Collection

Select suitable data for the project. This could include sign language videos, phrases, and other relevant data.

Data Processing

Perform exploratory data analysis on the collected data. Identify important landmarks in the sign language videos.

Modeling

Select an appropriate model for the project. Build the model and set it up for training. Train the model on the preprocessed data.

Evaluation

Evaluate the performance of the model. Tune the hyperparameters of the model to improve its performance.

Deploy Application

Develop a Sign Language website with a friendly user interface. Integrate the trained model with the website. Test the website and make sure it is working as expected.
